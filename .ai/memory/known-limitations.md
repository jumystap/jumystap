# Memory: Known limitations

- **CI/CD отсутствует.** Нет `.github/workflows`, `.gitlab-ci.yml`, Forge/Envoyer. Деплой — вручную
  или Docker. (TODO: добавить пайплайн pint+pest+build.)
- **Трекинг визитов синхронный.** `TrackVisits` пишет `Visit` синхронно в terminating-колбэке;
  `TrackVisitJob` существует, но постановка в очередь закомментирована. При росте трафика —
  кандидат на перевод в очередь.
- **config/database.php default = sqlite.** Расхождение с рабочим MySQL; полагаться только на
  `DB_CONNECTION` в `.env`.
- **Длинная сессия (~1 год).** `SESSION_LIFETIME=525600`. Учитывать при изменениях безопасности.
- **Смешанная админка.** Часть админ-экранов — Blade (`resources/views/admin/*`), часть — React
  (`Pages/Admin/*`). Нет единой карты соответствия.
- **Покрытие тестами частичное.** Есть тесты Auth, нескольких Services/Repositories/Controllers;
  большинство модулей (Ad, Resume, Certificate, Telegram, команды) не покрыты.
- **Локали — только ru/kz во фронте.** Несмотря на конвенцию `*_en` в БД, `public/locales/en` для
  react-i18next отсутствует; `ns` грузит ru/kz. Добавление en требует и locale-файлов.
- **Слабая типобезопасность фабрик.** Только `UserFactory`; тесты других сущностей требуют ручной
  подготовки данных.
- **`.env` обязателен и вне образа.** В Docker `.env` монтируется bind-маунтом; без него контейнеры
  не стартуют корректно.
