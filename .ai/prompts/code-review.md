# Prompt template: Code review

---

**Ревью изменений:** <ветка / диапазон диффа, напр. `master...HEAD`>

**Задача ревью:** проверить соответствие правилам JUMYSTAP, безопасность, корректность.
Только замечания — без правок без отдельного запроса.

**Опорные документы:** `DEVELOPMENT_RULES.md`, `.ai/context/security-rules.md`,
`.ai/agents/code-review.md`.

**Проверь:**
- Слоистость (тонкий контроллер, Service/Repository, нет SQL в контроллере).
- Валидация через FormRequest; enums вместо магических значений.
- Multilingual для переводимых полей; UI-строки во всех локалях + namespace.
- Схема — новой миграцией; FK/индексы.
- Нет лишних слоёв (Resources/Policies/Events/DTO) и UI-библиотек.
- Тесты в правильных каталогах; прогон зелёный.
- Безопасность: секреты, валидация ввода, санитизация вывода (dompurify).
- Несвязанная логика не затронута.

**Формат вывода:** список `файл:строка — проблема — рекомендация`; итоговый вердикт
(готово к мержу / требует правок).
